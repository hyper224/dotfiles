# dotfiles

Prerequisite

- `rsync`

Install dotfiles

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/hyper224/dotfiles/main/.tools/install.sh)"
```


Remove dotfiles

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/hyper224/dotfiles/main/.tools/uninstall/uninstall.sh)"
```
